# Простой анализатор логов nginx
Решение домашнего задания №1.

Пример запуска:
  - python log_analyzer.py -h
    - Параметры запуска:
        - -c (--config) <путь к конфигурационному файлу>
        - -l (--log) <путь для логов скрипта>

Запуск тестов:
  - python log_analyzer_tests.py -v
